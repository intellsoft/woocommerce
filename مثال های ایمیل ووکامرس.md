در ادامه مثال‌های عملی و کاربردی برای هر یک از هوک‌های بخش ایمیل‌های ووکامرس همراه با توضیح فارسی ارائه شده است.

## ۱. افزودن لوگوی فروشگاه به هدر ایمیل‌ها  
### هوک: `woocommerce_email_header`

```php
/**
 * قرار دادن لوگوی سفارشی به جای لوگوی پیش‌فرض در هدر ایمیل
 */
add_action( 'woocommerce_email_header', 'custom_email_logo', 10, 2 );
function custom_email_logo( $email_heading, $email ) {
	// حذف لوگوی پیش‌فرض (اختیاری)
	remove_action( 'woocommerce_email_header', 'woocommerce_email_header_image', 10 );
	
	$logo_url = get_stylesheet_directory_uri() . '/images/email-logo.png'; // آدرس لوگوی خود را بدهید
	echo '<div style="text-align:center; margin-bottom:20px;"><img src="' . esc_url( $logo_url ) . '" alt="' . get_bloginfo( 'name' ) . '" style="max-width:150px;"></div>';
}
```

## ۳. نمایش پیام تشکر قبل از جدول محصولات سفارش در ایمیل  
### هوک: `woocommerce_email_before_order_table`

```php
/**
 * یک متن تشکر بالای جدول محصولات در ایمیل سفارش اضافه می‌کند
 */
add_action( 'woocommerce_email_before_order_table', 'thank_you_message_before_table', 10, 4 );
function thank_you_message_before_table( $order, $sent_to_admin, $plain_text, $email ) {
	// فقط برای ایمیل‌های ارسالی به مشتری (نه مدیر)
	if ( ! $sent_to_admin ) {
		echo '<p style="font-size:16px; color:#4caf50;"><strong>از اعتماد شما سپاسگزاریم. سفارش شما در حال پیگیری است.</strong></p>';
	}
}
```

## ۴. درج کد رهگیری در پایین جدول محصولات (بعد از جدول)  
### هوک: `woocommerce_email_after_order_table`

```php
/**
 * نمایش کد رهگیری پس از جدول سفارش در ایمیل
 */
add_action( 'woocommerce_email_after_order_table', 'display_tracking_code_in_email', 15, 4 );
function display_tracking_code_in_email( $order, $sent_to_admin, $plain_text, $email ) {
	$tracking_code = $order->get_meta( 'tracking_code' );
	if ( ! empty( $tracking_code ) ) {
		echo '<div style="background:#f5f5f5; padding:10px; margin-top:20px;">';
		echo '<strong>کد رهگیری سفارش:</strong> ' . esc_html( $tracking_code );
		echo '</div>';
	}
}
```

## ۶. افزودن اطلاعات اضافی مشتری (مثلاً کد ملی) در بخش اطلاعات مشتری

add_action( 'woocommerce_email_customer_details', 'show_national_code_in_email', 10, 4 );

	}
}

## ۸. ثبت لاگ پس از ارسال موفق هر ایمیل  
### هوک: `woocommerce_email_sent`

```php
/**
 * ذخیرهٔ لاگ از ایمیل‌های ارسال شده برای بررسی در آینده
 */
add_action( 'woocommerce_email_sent', 'log_sent_emails', 10, 4 );
function log_sent_emails( $email_class, $email_id, $order_id, $email ) {
	if ( $order_id ) {
		$message = sprintf( 'ایمیل %s برای سفارش #%d ارسال شد.', $email_id, $order_id );
		error_log( $message );
	}
}
```

## ۹. تغییر موضوع ایمیل «تکمیل سفارش»  
### هوک: `woocommerce_email_subject_new_order` (نمونه برای `new_order`)

```php
/**
 * تغییر موضوع ایمیل سفارش جدید (ارسال به مدیر)
 */
add_filter( 'woocommerce_email_subject_new_order', 'change_new_order_email_subject', 10, 2 );
function change_new_order_email_subject( $subject, $order ) {
	return 'سفارش جدید شماره #' . $order->get_order_number() . ' – نیاز به بررسی';
}
```

**نمونه‌های دیگر `{class}`:**
- `woocommerce_email_subject_customer_processing_order` (ایمیل مشتری هنگام پردازش)
- `woocommerce_email_subject_customer_completed_order` (ایمیل مشتری پس از تکمیل)

## ۱۱. سفارشی‌سازی کامل استایل‌های ایمیل (اضافه کردن فونت فارسی)  
### هوک: `woocommerce_email_styles`

```php
/**
 * تغییر استایل‌های ایمیل: فونت وزیرمتن، رنگ لینک‌ها
 */
add_filter( 'woocommerce_email_styles', 'custom_email_styles' );
function custom_email_styles( $css ) {
	$custom_css = '
		@import url("https://cdn.fontcdn.ir/Font/Persian/Vazir/Vazir.css");
		#body_content, .td {
			font-family: "Vazir", Tahoma, sans-serif !important;
		}
		a {
			color: #e91e63 !important;
		}
	';
	return $css . $custom_css;
}
```
