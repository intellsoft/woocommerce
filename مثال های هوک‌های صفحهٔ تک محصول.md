در ادامه مثال‌های کاربردی برای هوک‌های صفحهٔ تک محصول ارائه شده است. این کدها را می‌توانید در `functions.php` قالب فرزند یا یک افزونهٔ اختصاصی اضافه کنید.

---

## ۱. افزودن پیام «محصول ویژه» در بالای صفحه محصول  
### هوک: `woocommerce_before_single_product`

```php
/**
 * قبل از محتوای اصلی صفحه محصول، یک نوار اطلاع‌رسانی نمایش می‌دهد
 */
add_action( 'woocommerce_before_single_product', 'special_product_message' );
function special_product_message() {
	global $product;
	if ( $product->is_featured() ) {
		echo '<div style="background:#fff3cd; padding:10px; margin-bottom:20px; text-align:center;">⭐ این محصول ویژه است! تا پایان هفته ارسال رایگان دارد.</div>';
	}
}
```

## ۲. اضافه کردن ویدیوی معرفی زیر گالری تصاویر  
### هوک: `woocommerce_before_single_product_summary`

```php
/**
 * بعد از گالری (قبل از خلاصه) یک ویدیو از محصول نمایش می‌دهد
 */
add_action( 'woocommerce_before_single_product_summary', 'show_product_video' );
function show_product_video() {
	global $product;
	$video_url = get_post_meta( $product->get_id(), 'product_video_url', true );
	if ( $video_url ) {
		echo '<div class="product-video" style="margin-bottom:20px;">';
		echo wp_oembed_get( $video_url );
		echo '</div>';
	}
}
```

## ۴. افزودن تایمر شمارش معکوس برای حراج  
### هوک: `woocommerce_before_add_to_cart_form`

```php
/**
 * قبل از فرم خرید، یک تایمر ساده برای اتمام حراج نمایش می‌دهد
 */
add_action( 'woocommerce_before_add_to_cart_form', 'sale_countdown_timer' );
function sale_countdown_timer() {
	global $product;
	if ( $product->is_on_sale() ) {
		$sale_to = $product->get_date_on_sale_to();
		if ( $sale_to ) {
			echo '<div class="sale-timer" data-sale-end="' . esc_attr( $sale_to->getTimestamp() ) . '" style="background:#f00;color:#fff;padding:8px;margin-bottom:15px;text-align:center;">
				⏳ این تخفیف تا <span id="countdown"></span> دیگر معتبر است
			</div>';
			// برای تکمیل، نیاز به یک اسکریپت جاوااسکریپت دارید، اما این بخش هوک را نشان می‌دهد
		}
	}
}
```

## ۱۰. افزودن یک تب جدید «نقد و بررسی تخصصی»  
### هوک: `woocommerce_product_tabs` (Filter)

```php
/**
 * یک تب جدید با محتوای دلخواه به صفحه محصول اضافه می‌کند
 */
add_filter( 'woocommerce_product_tabs', 'add_expert_review_tab' );
function add_expert_review_tab( $tabs ) {
	$tabs['expert_review'] = array(
		'title'    => 'نقد تخصصی',
		'priority' => 15, // بین توضیحات (10) و نظرات (20)
		'callback' => 'expert_review_tab_content'
	);
	return $tabs;
}

function expert_review_tab_content() {
	echo '<h2>نقد تخصصی این محصول</h2>';
	echo '<p>در این بخش کارشناسان ما به بررسی کامل محصول پرداخته‌اند.</p>';
}
```

## ۱۵. تغییر کامل HTML قیمت (افزودن عبارت «تومان» و قالب‌بندی)  
### هوک: `woocommerce_get_price_html`

```php
/**
 * نمایش قیمت با پسوند تومان و در صورت حراج، قیمت اصلی خط‌خورده شود
 */
add_filter( 'woocommerce_get_price_html', 'custom_price_format', 10, 2 );
function custom_price_format( $price, $product ) {
	if ( $product->is_on_sale() ) {
		$regular = wc_price( $product->get_regular_price() );
		$sale    = wc_price( $product->get_sale_price() );
		return '<del>' . $regular . '</del> <ins>' . $sale . ' تومان</ins>';
	}
	return $price . ' تومان';
}
```

## ۱۶. نمایش درصد تخفیف روی محصولات حراجی در صفحه محصول  
### هوک: `woocommerce_before_single_product_summary` یا `woocommerce_single_product_summary` (اولویت ۱۰)

```php
/**
 * یک برچسب درصد تخفیف کنار قیمت یا بالای تصویر اضافه می‌کند
 */
add_action( 'woocommerce_before_single_product_summary', 'sale_percentage_badge', 15 );
function sale_percentage_badge() {
	global $product;
	if ( $product->is_on_sale() ) {
		$regular = $product->get_regular_price();
		$sale    = $product->get_sale_price();
		if ( $regular > 0 ) {
			$percent = round( 100 - ( $sale / $regular * 100 ) );
			echo '<span style="position:absolute; top:10px; right:10px; background:red; color:#fff; padding:5px 10px;">' . $percent . '% تخفیف</span>';
		}
	}
}
```
