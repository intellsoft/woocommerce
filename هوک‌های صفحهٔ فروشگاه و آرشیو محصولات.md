در ادامه برای هر یک از هوک‌های جدول، مثال‌های عملی و کاربردی همراه با توضیح فارسی آورده شده است. این کدها را می‌توانید در فایل `functions.php` قالب فرزند یا یک افزونهٔ اختصاصی قرار دهید.

---

## ۱. افزودن بنر تبلیغاتی قبل از لیست محصولات  
### هوک: `woocommerce_before_shop_loop`

```php
/**
 * نمایش یک بنر تخفیف در بالای لیست محصولات
 */
add_action( 'woocommerce_before_shop_loop', 'add_promo_banner' );
function add_promo_banner() {
	// فقط در صفحهٔ اصلی فروشگاه یا صفحات دسته‌بندی
	if ( is_shop() || is_product_category() ) {
		echo '<div class="promo-banner" style="background:#f8d7da; padding:15px; text-align:center; margin-bottom:20px;">';
		echo '🔥 ارسال رایگان برای سفارش‌های بالای ۵۰۰ هزار تومان';
		echo '</div>';
	}
}
```

## ۴. اضافه کردن نشان «جدید» روی تصویر محصول در آرشیو  
### هوک: `woocommerce_before_shop_loop_item_title`

```php
/**
 * در کارت محصول، قبل از عنوان (بالای تصویر) یک برچسب "جدید" اضافه می‌کند
 */
add_action( 'woocommerce_before_shop_loop_item_title', 'show_new_badge', 5 );
function show_new_badge() {
	global $product;
	// شرط دلخواه: مثلاً اگر محصول در ۳۰ روز اخیر منتشر شده باشد
	$newness_days = 30;
	$created = strtotime( $product->get_date_created() );
	if ( ( time() - $created ) < ( $newness_days * 24 * 60 * 60 ) ) {
		echo '<span class="badge new-badge" style="position:absolute; top:10px; left:10px; background:#28a745; color:white; padding:5px 10px; z-index:10;">جدید</span>';
	}
}
```

## ۶. نمایش متن «تمام شد» برای محصولات ناموجود در حلقه  
### هوک: `woocommerce_after_shop_loop_item`

```php
/**
 * بعد از بسته شدن کارت محصول، اگر ناموجود بود پیغام بدهد
 */
add_action( 'woocommerce_after_shop_loop_item', 'out_of_stock_message', 5 );
function out_of_stock_message() {
	global $product;
	if ( ! $product->is_in_stock() ) {
		echo '<p class="stock out-of-stock" style="color:red; text-align:center;">ناموجود</p>';
	}
}
```

## ۷. حذف ابزار مرتب‌سازی و شمارش نتایج از بالای لیست  
### هوک: `woocommerce_before_shop_loop`

این ابزارها به‌طور پیش‌فرض در اولویت‌های مختلف به این هوک متصل شده‌اند. برای حذفشان:

```php
/**
 * حذف کامل ابزار مرتب‌سازی و نمایش تعداد نتایج
 */
remove_action( 'woocommerce_before_shop_loop', 'woocommerce_catalog_ordering', 30 );
remove_action( 'woocommerce_before_shop_loop', 'woocommerce_result_count', 20 );
```

## ۸. تغییر HTML دکمهٔ افزودن به سبد (اضافه کردن آیکون)  
### هوک: `woocommerce_loop_add_to_cart_link`

```php
/**
 * یک آیکون سبد خرید به دکمه اضافه می‌کند
 */
add_filter( 'woocommerce_loop_add_to_cart_link', 'add_icon_to_add_to_cart_button', 10, 2 );
function add_icon_to_add_to_cart_button( $button, $product ) {
	$icon = '<span class="dashicons dashicons-cart" style="vertical-align:middle;"></span> ';
	return str_replace( '>', '>' . $icon, $button ); // آیکون داخل تگ <a> قرار می‌گیرد
}
```

## ۱۰. افزودن متن «فروش ویژه» کنار قیمت حراجی  
### هوک: `woocommerce_after_shop_loop_item_title` (قبلاً ذکر شد)  
### فیلتر: `woocommerce_get_price_html`

```php
/**
 * در کنار قیمت حراجی عبارت «فروش ویژه» اضافه می‌شود
 */
add_filter( 'woocommerce_get_price_html', 'add_sale_flash_text', 10, 2 );
function add_sale_flash_text( $price, $product ) {
	if ( $product->is_on_sale() ) {
		$price .= ' <span class="sale-flash">(فروش ویژه)</span>';
	}
	return $price;
}
```

## ۱۲. اضافه کردن یک کلاس CSS به کارت محصول بر اساس دسته‌بندی  
### هوک: `woocommerce_before_shop_loop_item`

```php
/**
 * کلاس سفارشی به عنصر <li> اضافه می‌شود (با فیلتر post_class)
 * اما از طریق این هوک هم می‌توان دستکاری کرد. روش بهتر استفاده از فیلتر post_class است:
 */
add_filter( 'post_class', 'add_category_specific_class', 10, 3 );
function add_category_specific_class( $classes, $class, $post_id ) {
	if ( 'product' === get_post_type( $post_id ) ) {
		$categories = wp_get_post_terms( $post_id, 'product_cat' );
		foreach ( $categories as $cat ) {
			$classes[] = 'product-cat-' . $cat->slug;
		}
	}
	return $classes;
}
```
این کد برای هر محصول کلاسی مانند `product-cat-phone` اضافه می‌کند و می‌توان استایل‌های متفاوتی به دسته‌ها داد.

این مثال‌ها پایه‌ای برای سفارشی‌سازی صفحات فروشگاه و آرشیو هستند. با ترکیب این هوک‌ها می‌توانید چیدمان و محتوای این صفحات را کاملاً مطابق نیاز خود تغییر دهید.
