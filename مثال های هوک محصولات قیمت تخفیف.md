در این بخش مثال‌های کاربردی برای تمام هوک‌های مربوط به محصولات، قیمت‌ها و تخفیف‌ها ارائه می‌شود.

## Action Hooks

### ۱. `woocommerce_product_options_general_product_data`  
**افزودن فیلد سفارشی به بخش «داده‌های عمومی» محصول**

```php
/**
 * اضافه کردن فیلد "زمان آماده‌سازی" به برگهٔ عمومی ویرایش محصول
 */
add_action( 'woocommerce_product_options_general_product_data', 'add_preparation_time_field' );
function add_preparation_time_field() {
	echo '<div class="options_group">';
	woocommerce_wp_text_input( array(
		'id'          => '_preparation_time',
		'label'       => 'زمان آماده‌سازی (روز)',
		'placeholder' => 'مثلاً ۳',
		'type'        => 'number',
		'desc_tip'    => true,
		'description' => 'مدت زمان لازم تا ارسال محصول',
	) );
	echo '</div>';
}
```

### ۲. `woocommerce_process_product_meta`  
**ذخیره‌سازی فیلد سفارشی هنگام ذخیرهٔ محصول**

```php
/**
 * ذخیره کردن مقدار فیلد "زمان آماده‌سازی" در متادیتای محصول
 */
add_action( 'woocommerce_process_product_meta', 'save_preparation_time_field' );
function save_preparation_time_field( $post_id ) {
	$preparation_time = isset( $_POST['_preparation_time'] ) ? intval( $_POST['_preparation_time'] ) : '';
	update_post_meta( $post_id, '_preparation_time', $preparation_time );
}
```

## Filter Hooks

### ۳. `woocommerce_get_price_html`  
**نمایش عبارت «رایگان» برای محصولات با قیمت صفر و تغییر قالب قیمت**

```php
/**
 * اگر قیمت صفر بود، "رایگان" نشان بده، در غیر این‌صورت "تومان" اضافه کن
 */
add_filter( 'woocommerce_get_price_html', 'custom_price_display', 10, 2 );
function custom_price_display( $price_html, $product ) {
	if ( $product->get_price() == 0 ) {
		return '<span class="free-price">رایگان</span>';
	}
	// جایگزین نماد ارز پیش‌فرض با "تومان"
	return str_replace( '&nbsp;', '', $price_html ) . ' تومان';
}
```

### ۴. `woocommerce_product_get_price`  
**اعمال تخفیف عمومی روی قیمت (مثلاً ۱۰٪ تخفیف روی همهٔ محصولات)**

```php
/**
 * کم کردن ۱۰ درصد از قیمت فعلی محصول
 */
add_filter( 'woocommerce_product_get_price', 'apply_global_discount', 99, 2 );
function apply_global_discount( $price, $product ) {
	if ( is_admin() ) return $price; // در پنل ادمین تغییر نکند
	return $price * 0.9;
}
```

### ۵. `woocommerce_product_get_regular_price`  
**تغییر قیمت اصلی فقط برای نمایش (بدون تأثیر روی سبد خرید)**

```php
/**
 * اگر محصول در یک دسته خاص بود، قیمت اصلی را ۲۰٪ بیشتر نشان بده (تا تخفیف جذابتر شود)
 */
add_filter( 'woocommerce_product_get_regular_price', 'inflate_regular_price', 99, 2 );
function inflate_regular_price( $regular_price, $product ) {
	if ( has_term( 'ویژه', 'product_cat', $product->get_id() ) ) {
		return $regular_price * 1.2;
	}
	return $regular_price;
}
```

### ۶. `woocommerce_product_get_sale_price`  
**محدود کردن قیمت حراج به عددی خاص (حداقل قیمت فروش)**

```php
/**
 * اگر قیمت حراج کمتر از ۱۰۰۰۰ تومان بود، آن را به ۱۰۰۰۰ تغییر بده
 */
add_filter( 'woocommerce_product_get_sale_price', 'min_sale_price_limit', 99, 2 );
function min_sale_price_limit( $sale_price, $product ) {
	if ( $sale_price && $sale_price < 10000 ) {
		return 10000;
	}
	return $sale_price;
}
```

### ۷. `woocommerce_get_price_excluding_tax`  
**اعمال هزینهٔ اضافی روی قیمت بدون مالیات**

```php
/**
 * ۵٪ هزینهٔ بیمه به قیمت بدون مالیات اضافه شود (فقط در front-end)
 */
add_filter( 'woocommerce_get_price_excluding_tax', 'add_insurance_fee', 10, 4 );
function add_insurance_fee( $price, $product, $qty, $tax_class ) {
	if ( is_admin() ) return $price;
	return $price * 1.05;
}
```

### ۸. `woocommerce_get_price_including_tax`  
**گرد کردن قیمت با مالیات به نزدیکترین هزار تومان**

```php
/**
 * قیمت نهایی را به نزدیکترین ۱۰۰۰ تومان گرد کن
 */
add_filter( 'woocommerce_get_price_including_tax', 'round_price_to_thousand', 10, 4 );
function round_price_to_thousand( $price, $product, $qty, $tax_class ) {
	return round( $price / 1000 ) * 1000;
}
```

### ۹. `woocommerce_currency_symbol`  
**تغییر نماد ارز به «تومان» با فونت زیباتر**

```php
/**
 * جایگزینی نماد ارز با عبارت "تومان"
 */
add_filter( 'woocommerce_currency_symbol', 'change_currency_symbol_to_toman' );
function change_currency_symbol_to_toman( $symbol ) {
	return 'تومان';
}
```

### ۱۰. `woocommerce_format_sale_price`  
**تغییر ظاهر محدودهٔ قیمت حراج (مثلاً افزودن درصد تخفیف)**

```php
/**
 * نمایش درصد تخفیف کنار قیمت حراج
 */
add_filter( 'woocommerce_format_sale_price', 'display_sale_percentage', 10, 3 );
function display_sale_percentage( $price, $regular_price, $sale_price ) {
	if ( $regular_price > 0 ) {
		$percentage = round( ( ( $regular_price - $sale_price ) / $regular_price ) * 100 );
		return $price . ' <span class="sale-percent">(' . $percentage . '% تخفیف)</span>';
	}
	return $price;
}
```

### ۱۱. `woocommerce_product_is_in_stock`  
**علامت‌گذاری دستی برخی محصولات به‌عنوان ناموجود (بر اساس دسته)**

```php
/**
 * اگر محصول در دستهٔ "پیش‌فروش" باشد، آن را ناموجود نشان بده
 */
add_filter( 'woocommerce_product_is_in_stock', 'make_preorder_products_out_of_stock', 10, 2 );
function make_preorder_products_out_of_stock( $is_in_stock, $product ) {
	if ( has_term( 'پیش‌فروش', 'product_cat', $product->get_id() ) ) {
		return false;
	}
	return $is_in_stock;
}
```

### ۱۲. `woocommerce_product_get_stock_quantity`  
**نمایش تعداد موجودی به‌صورت تقریبی (گرد شده)**

```php
/**
 * برای محصولات با موجودی بالا، تعداد را به صورت "بیش از ۱۰ عدد" نشان بده
 */
add_filter( 'woocommerce_product_get_stock_quantity', 'approximate_stock_quantity', 99, 2 );
function approximate_stock_quantity( $quantity, $product ) {
	if ( $quantity > 10 ) {
		return 'بیش از ۱۰ عدد';
	}
	return $quantity;
}
```

### ۱۳. `woocommerce_product_get_stock_status`  
**تغییر وضعیت موجودی محصول به «پیش‌فروش» در صورت وجود برچسب خاص**

```php
/**
 * اگر محصول برچسب "پیش‌فروش" داشته باشد، وضعیت را به "پیش‌فروش" تغییر بده
 */
add_filter( 'woocommerce_product_get_stock_status', 'set_preorder_stock_status', 99, 2 );
function set_preorder_stock_status( $status, $product ) {
	if ( has_term( 'پیش‌فروش', 'product_cat', $product->get_id() ) ) {
		return 'onbackorder'; // وضعیت "پیش‌فروش" (موجود نیست اما می‌توان سفارش داد)
	}
	return $status;
}
```

### ۱۴. `woocommerce_get_availability`  
**سفارشی‌سازی متن و کلاس موجودی نمایش داده شده**

```php
/**
 * برای محصولات با موجودی کم، متن هشداردهنده و کلاس قرمز بده
 */
add_filter( 'woocommerce_get_availability', 'custom_availability_message', 99, 2 );
function custom_availability_message( $availability, $product ) {
	if ( $product->is_in_stock() && $product->get_stock_quantity() <= 3 ) {
		$availability['availability'] = 'فقط ' . $product->get_stock_quantity() . ' عدد در انبار باقی مانده';
		$availability['class'] = 'low-stock';
	}
	return $availability;
}
```
