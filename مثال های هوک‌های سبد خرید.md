در این بخش، مثال‌های کاربردی برای شخصی‌سازی صفحهٔ سبد خرید و فرایندهای مرتبط با آن بر اساس هوک‌های اکشن و فیلتر ارائه می‌شود. تمام کدها در فایل `functions.php` قالب فرزند یا یک افزونهٔ اختصاصی قرار می‌گیرند.

---

## 🛒 هوک‌های سبد خرید – مثال‌های عملی

### ۱. نمایش پیام تخفیف در بالای صفحه سبد خرید  
#### هوک: `woocommerce_before_cart`

```php
/**
 * نمایش یک پیام اطلاع‌رسانی قبل از محتوای اصلی سبد خرید
 */
add_action( 'woocommerce_before_cart', 'cart_page_promo_message' );
function cart_page_promo_message() {
    $cart_total = WC()->cart->get_subtotal();
    if ( $cart_total < 500000 ) {
        echo '<div class="woocommerce-info">فقط ' . number_format( 500000 - $cart_total ) . ' تومان تا ارسال رایگان فاصله دارید!</div>';
    }
}
```

### ۳. اضافه کردن یک ردیف هشدار قبل از محصولات در سبد  
#### هوک: `woocommerce_before_cart_contents`

```php
/**
 * اگر سبد شامل یک محصول خاص باشد، هشدار نمایش بده
 */
add_action( 'woocommerce_before_cart_contents', 'warn_specific_product_in_cart' );
function warn_specific_product_in_cart() {
    $target_product_id = 123; // شناسه محصول مورد نظر
    foreach ( WC()->cart->get_cart() as $cart_item ) {
        if ( $cart_item['product_id'] == $target_product_id ) {
            echo '<tr><td colspan="6" class="cart-warning" style="background:#fff3cd; padding:10px;">⚠️ این محصول فقط تا ۲ عدد مجاز است. لطفاً تعداد را کاهش دهید.</td></tr>';
            break;
        }
    }
}
```

### ۴. افزودن یک ستون «هدیه» درون جدول سبد خرید  
#### هوک: `woocommerce_cart_contents` (برای افزودن هدر ستون و سلول)

برای افزودن یک ستون کامل (هدر + سلول) نیاز به دو بخش داریم: یکی برای سرستون‌ها با فیلتر `woocommerce_checkout_cart_item_quantity` یا استفاده از هوک `woocommerce_before_cart_contents` برای تغییر هدر، اما راه استاندارد این است که فیلترهای مربوط به هر آیتم را تغییر دهیم. در این مثال، از طریق هوک‌های مختلف، یک چک‌باکس «بسته‌بندی کادو» اضافه می‌کنیم.

```php
// ۱. افزودن سرستون "بسته‌بندی کادو" به هدر جدول
add_action( 'woocommerce_before_cart_contents', 'add_gift_wrap_column_header' );
function add_gift_wrap_column_header() {
    echo '<th class="gift-wrap-col">بسته‌بندی کادو</th>';
}

// ۲. افزودن چک‌باکس در هر ردیف محصول
add_action( 'woocommerce_after_cart_item_name', 'add_gift_wrap_checkbox', 10, 2 );
function add_gift_wrap_checkbox( $cart_item, $cart_item_key ) {
    echo '<td class="gift-wrap-col"><input type="checkbox" name="gift_wrap[' . $cart_item_key . ']" value="1" ' . checked( isset($cart_item['gift_wrap']) && $cart_item['gift_wrap'], true, false ) . ' /> بپیچ</td>';
}
```
توجه: این روش نیازمند تغییرات CSS برای نمایش صحیح ستون است و برای ذخیرهٔ انتخاب باید از `woocommerce_add_cart_item_data` نیز استفاده کرد.

### ۵. نمایش دکمهٔ «خالی کردن سبد خرید» بعد از محصولات  
#### هوک: `woocommerce_after_cart_contents`

```php
/**
 * یک دکمه برای خالی کردن کامل سبد خرید در پایین جدول اضافه می‌کند
 */
add_action( 'woocommerce_after_cart_contents', 'add_empty_cart_button' );
function add_empty_cart_button() {
    echo '<tr><td colspan="6"><a href="' . esc_url( add_query_arg( 'empty-cart', 'true', wc_get_cart_url() ) ) . '" class="button empty-cart-button">خالی کردن سبد</a></td></tr>';
}

// پردازش درخواست خالی کردن سبد
add_action( 'init', 'handle_empty_cart_action' );
function handle_empty_cart_action() {
    if ( isset( $_GET['empty-cart'] ) && 'true' === $_GET['empty-cart'] ) {
        WC()->cart->empty_cart();
        wp_redirect( wc_get_cart_url() );
        exit;
    }
}
```

### ۱۴. افزودن شناسه (SKU) به نام محصول در سبد  
#### هوک: `woocommerce_cart_item_name` (فیلتر)

```php
/**
 * کنار نام محصول، کد SKU را هم نمایش بده
 */
add_filter( 'woocommerce_cart_item_name', 'add_sku_to_cart_item_name', 10, 3 );
function add_sku_to_cart_item_name( $name, $cart_item, $cart_item_key ) {
    $product = $cart_item['data'];
    if ( $product->get_sku() ) {
        $name .= ' <small>(کد: ' . $product->get_sku() . ')</small>';
    }
    return $name;
}
```

### ۱۹. اضافه کردن یک فیلد دلخواه (مثلاً حکاکی) به محصول در سبد  
#### هوک: `woocommerce_add_cart_item_data` (فیلتر)

```php
/**
 * ذخیرهٔ متن حکاکی هنگام افزودن به سبد
 */
add_filter( 'woocommerce_add_cart_item_data', 'add_engraving_text_to_cart_item', 10, 2 );
function add_engraving_text_to_cart_item( $cart_item_data, $product_id ) {
    if ( isset( $_POST['engraving_text'] ) && ! empty( $_POST['engraving_text'] ) ) {
        $cart_item_data['engraving_text'] = sanitize_text_field( $_POST['engraving_text'] );
    }
    return $cart_item_data;
}

// نمایش متن حکاکی در سبد
add_filter( 'woocommerce_get_item_data', 'display_engraving_text_in_cart', 10, 2 );
function display_engraving_text_in_cart( $item_data, $cart_item ) {
    if ( isset( $cart_item['engraving_text'] ) ) {
        $item_data[] = array(
            'name'  => 'حکاکی',
            'value' => $cart_item['engraving_text'],
        );
    }
    return $item_data;
}
```

### ۲۱. سفارشی‌سازی برچسب روش‌های حمل و نقل در سبد  
#### هوک: `woocommerce_cart_shipping_method_full_label` (فیلتر)

```php
/**
 * نام روش حمل و نقل را با توضیح اضافه نمایش بده
 */
add_filter( 'woocommerce_cart_shipping_method_full_label', 'custom_shipping_label', 10, 2 );
function custom_shipping_label( $label, $method ) {
    if ( $method->method_id == 'flat_rate' ) {
        $label = $label . ' (تحویل ۳-۴ روزه)';
    }
    return $label;
}
```
