در ادامه برای تمام هوک‌های بخش تسویه حساب که در جدول ذکر کردید، مثال‌های عملی و کاربردی همراه با توضیح فارسی ارائه می‌شود. این کدها را می‌توانید در `functions.php` قالب فرزند یا یک افزونهٔ اختصاصی قرار دهید.

---

## 🛒 **Action Hooks (اکشن‌ها)**

### ۱. `woocommerce_before_checkout_form`
**توضیح:** قبل از کل فرم تسویه حساب اجرا می‌شود. مناسب برای افزودن پیام خوش‌آمد، بنر یا دستورالعمل‌های مهم.

```php
/**
 * نمایش یک پیام راهنما در بالای فرم تسویه حساب
 */
add_action( 'woocommerce_before_checkout_form', 'checkout_welcome_message' );
function checkout_welcome_message() {
    echo '<div class="checkout-notice" style="background:#e7f3ff; padding:15px; margin-bottom:20px; border-radius:5px;">';
    echo '📦 لطفاً آدرس خود را با دقت وارد کنید. سفارش شما پس از تأیید نهایی ارسال خواهد شد.';
    echo '</div>';
}
```

### ۲. `woocommerce_checkout_before_customer_details`
**توضیح:** قبل از فیلدهای اطلاعات مشتری (صورتحساب و حمل). می‌توانید راهنمایی یا هشدار خاصی قرار دهید.

```php
/**
 * افزودن یک هشدار قبل از وارد کردن اطلاعات شخصی
 */
add_action( 'woocommerce_checkout_before_customer_details', 'privacy_notice_before_details' );
function privacy_notice_before_details() {
    echo '<p style="font-size:0.9em; color:#666;">🔒 اطلاعات شما کاملاً محرمانه باقی می‌ماند و فقط برای ارسال سفارش استفاده می‌شود.</p>';
}
```

### ۳. `woocommerce_checkout_billing`
**توضیح:** درون بخش فیلدهای صورتحساب. ایده‌آل برای افزودن فیلدهای سفارشی اضافی (مانند کد ملی، نام شرکت و …).

```php
/**
 * اضافه کردن یک فیلد "نام شرکت" به بخش صورتحساب
 */
add_action( 'woocommerce_checkout_billing', 'add_company_field_to_billing' );
function add_company_field_to_billing() {
    woocommerce_form_field( 'billing_company', array(
        'type'        => 'text',
        'class'       => array('form-row-wide'),
        'label'       => 'نام شرکت (اختیاری)',
        'placeholder' => 'در صورت خرید سازمانی',
    ), WC()->checkout->get_value( 'billing_company' ) );
}
```

/**

### ۱۴. `woocommerce_checkout_process`
**توضیح:** هنگام پردازش و اعتبارسنجی فرم (بعد از کلیک ثبت سفارش). عالی برای اعتبارسنجی فیلدهای سفارشی و نمایش خطا.

```php
/**
 * اعتبارسنجی فیلد سفارشی "نام شرکت" (نباید خالی باشد اگر پر شده)
 */
add_action( 'woocommerce_checkout_process', 'validate_custom_fields' );
function validate_custom_fields() {
    if ( isset( $_POST['billing_company'] ) && ! empty( $_POST['billing_company'] ) ) {
        // اگر پر شده، باید حداقل ۳ کاراکتر باشد
        if ( strlen( $_POST['billing_company'] ) < 3 ) {
            wc_add_notice( 'نام شرکت باید حداقل ۳ کاراکتر داشته باشد.', 'error' );
        }
    }
}
```

### ۱۵. `woocommerce_checkout_update_order_review`
**توضیح:** هنگام به‌روزرسانی بخش مرور سفارش با Ajax (مثلاً تغییر آدرس یا روش حمل). می‌توان برای افزودن هزینه‌های پویا استفاده کرد.

```php
/**
 * افزودن هزینه بسته‌بندی در صورت انتخاب آپشن
 */
add_action( 'woocommerce_cart_calculate_fees', 'add_gift_wrap_fee' );
function add_gift_wrap_fee() {
    if ( isset( $_POST['post_data'] ) ) {
        parse_str( $_POST['post_data'], $post_data );
        if ( isset( $post_data['gift_wrap'] ) && $post_data['gift_wrap'] === 'yes' ) {
            WC()->cart->add_fee( 'بسته‌بندی کادو', 50000 );
        }
    }
}
// اطمینان از به‌روزرسانی مبالغ هنگام تغییر آپشن
add_action( 'woocommerce_checkout_update_order_review', 'refresh_gift_wrap_fee' );
function refresh_gift_wrap_fee( $post_data ) {
    WC()->cart->calculate_fees();
}
```

## 🎛️ **Filter Hooks (فیلترها)**

### ۱۹. `woocommerce_checkout_fields`
**توضیح:** فیلتر کل فیلدهای فرم تسویه حساب (بیلینگ، شیپینگ و فیلدهای اضافی). قدرتمندترین فیلتر برای مدیریت فیلدها.

```php
/**
 * حذف فیلد "نام شرکت" و غیرضروری کردن فیلد تلفن، و تغییر لیبل نام
 */
add_filter( 'woocommerce_checkout_fields', 'customize_all_checkout_fields' );
function customize_all_checkout_fields( $fields ) {
    // حذف کامل فیلد شرکت از بیلینگ
    unset( $fields['billing']['billing_company'] );
    // غیرضروری کردن تلفن
    $fields['billing']['billing_phone']['required'] = false;
    // تغییر لیبل نام کوچک
    $fields['billing']['billing_first_name']['label'] = 'نام شما';
    return $fields;
}
```

---

### ۲۰. `woocommerce_billing_fields`
**توضیح:** فیلتر فقط فیلدهای صورتحساب. مشابه بالا ولی فقط برای بیلینگ.

```php
/**
 * اضافه کردن یک فیلد "کد ملی" به بخش صورتحساب و اجباری کردن آن
 */
add_filter( 'woocommerce_billing_fields', 'add_national_id_field' );
function add_national_id_field( $fields ) {
    $fields['billing_national_id'] = array(
        'label'       => 'کد ملی',
        'required'    => true,
        'class'       => array('form-row-wide'),
        'priority'    => 25,
        'placeholder' => 'شماره ۱۰ رقمی',
        'type'        => 'text',
    );
    return $fields;
}
```

### ۲۳. `woocommerce_checkout_get_value`
**توضیح:** مقدار پیش‌فرض هر فیلد تسویه حساب را تغییر می‌دهد.

```php
/**
 * پر کردن خودکار فیلد "توضیحات تحویل" برای کاربران مهمان
 */
add_filter( 'woocommerce_checkout_get_value', 'default_delivery_note', 10, 2 );
function default_delivery_note( $value, $input ) {
    if ( 'delivery_note' === $input && ! is_user_logged_in() ) {
        return 'لطفاً قبل از تحویل تماس بگیرید.';
    }
    return $value;
}
```

### ۲۵. `woocommerce_available_payment_gateways`
**توضیح:** لیست گیت‌وی‌های پرداخت قابل نمایش را فیلتر می‌کند. می‌توان بر اساس مبلغ سبد، برخی درگاه‌ها را مخفی کرد.

```php
/**
 * مخفی کردن درگاه "پرداخت در محل" برای سفارش‌های بالای ۱ میلیون تومان
 */
add_filter( 'woocommerce_available_payment_gateways', 'disable_cod_for_high_amount' );
function disable_cod_for_high_amount( $gateways ) {
    if ( WC()->cart && WC()->cart->total > 1000000 ) {
        unset( $gateways['cod'] );
    }
    return $gateways;
}
```

### ۲۶. `woocommerce_cod_process_payment_order_status`
**توضیح:** وضعیت سفارش را برای روش پرداخت در محل (Cash on Delivery) تعیین می‌کند. پیش‌فرض `processing` است، می‌توانید به `on-hold` تغییر دهید.

```php

 * سفارشات پرداخت در محل را به جای "در حال پردازش" به "در انتظار" ببرید
 */
add_filter( 'woocommerce_cod_process_payment_order_status', 'change_cod_order_status' );
function change_cod_order_status( $status ) {
    return 'on-hold'; // وضعیت "در انتظار"
}
```
